wordpress建站攻略

Posted on 2017-07-17 Edited on 2019-08-28 In feat , wordpress Views: Valine:

环境配置要求

  • PHP 5.2.4或更高版本
  • MySQL 5.0或更高版本

建站关键过程

  • 中文官网安装教程

  • 通过shell安装wordpress,mysql,php5,php5-fpm,php-mysql,nginx

  • apt-get命令详解

  • nginx修改配置(须配置php)

    server {

    listen 80 default_server;
listen [::]:80 default_server ipv6only=on;

root /var/www/html/wordpress;
index index.html index.htm index.php;

error_log /var/log/wp-error.log;
# Make site accessible from http://localhost/
server_name blog.qingf.me;

location / {
    try_files $uri $uri/ =404;
}
location ~ \.php$ {
    fastcgi_pass 127.0.0.1:9000;
    fastcgi_index index.php;
    fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    include fastcgi_params;
 }

    }

建站结果

  • web访问此目录进入建站初始化配置页面即成功。

建站配置
